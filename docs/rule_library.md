# Geopolitical Rule Library

This document outlines the symbolic geopolitical rules utilized by the Geo-Sim framework.

### Nuclear
* `NUC_001`: Nuclear Deterrence Baseline. Direct military attack on a nuclear-armed state raises nuclear escalation probability.
* `NUC_002`: Mutual Assured Destruction Stability. When both sides have second-strike capability, full-scale war probability drops.

### Economic
* `ECO_001`: Resource Exporter Sanctions Resilience. Countries with significant resource exports resist financial sanctions better.
* `ECO_002`: SWIFT Exclusion Economic Shock. Exclusion from SWIFT causes immediate financial system stress.

*Note: More rules are generated dynamically by the Neuro-Symbolic Learning Engine.*
